Look Around You
===============

    Acorn DFS

    BASIC

    >10 PRINT "LOOK AROUND YOU ";
    >20 GOTO 10
    > RUN

## Installation

    npm install -g lookaroundyou

## Usage

    lookaroundyou HELLO WORLD

or

    lookaroundyou 'THANK YOU, ANTS. '

or

    lookaroundyou --text='tick ' --delay=1
